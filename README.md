# FinTrack 💰

A modern expense tracker web app with dashboard, charts, and full CRUD.

## Features
- Add, edit, delete expenses
- Categories: Food, Transport, Shopping, Entertainment, Health, Education, Bills, Other
- Dashboard with summary cards
- Monthly bar chart & category pie chart
- Analytics page with trend line & doughnut chart
- Search & filter by category/month
- Dark theme, responsive design

## Tech Stack
- **Frontend:** HTML, CSS, Vanilla JS, Chart.js
- **Backend:** Node.js + Express
- **Database:** JSON file (no setup needed)

## Run Locally

```bash
npm install
npm start
```

Open http://localhost:3000

## Deploy to Render (Free)

1. Push to GitHub
2. Go to [render.com](https://render.com) → New Web Service
3. Connect your repo
4. Build command: `npm install`
5. Start command: `node server.js`
6. Done! 🚀
